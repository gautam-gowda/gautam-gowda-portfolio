Digital Traffic Signal Controller using Moore FSM with Emergency Override
Verilog HDLIcarus Verilog GTKWaveMoore FSMTestbench
Overview

A synchronous 2-way traffic signal controller using Moore FSM in Verilog. Enforces deterministic timing and supports real-time emergency vehicle override with priority arbitration.

Key Design Decisions
Moore architecture — outputs depend only on state, not inputs. Ensures output stability.
Emergency override forces S0 (Road A) or S2 (Road B) from any state asynchronously.
Road A given priority when both emergency signals are simultaneously active.
Counter resets on every state transition — Green=10 cycles, Yellow=4 cycles.
Testbench verified: reset, normal FSM cycle, emergency A, emergency B, simultaneous override.

ML-Based Landslide Susceptibility Mapping — Western Ghats, Karnataka
IEEE PAPER TITLE
"A Geospatial Machine Learning Framework for Landslide Susceptibility Assessment Using Historical Inventories and Spatial Block Validation"
Research Overview

A machine learning pipeline for geospatial landslide risk prediction across Karnataka's Western Ghats. Built on a custom multi-source dataset integrating satellite imagery, rainfall records, and DEM-derived terrain features. Three models compared — Random Forest, XGBoost, and Logistic Regression — with spatial block cross-validation to prevent data leakage across geographic regions.

Susceptibility Map — Western Ghats Output

The output map classifies regions into 5 susceptibility zones — Very Low to Very High — using the Random Forest model with spatial block validation. Darker red zones indicate highest landslide risk, concentrated along steep river valleys.

SHAP Feature Importance Analysis

SHAP (SHapley Additive exPlanations) reveals river proximity (dist_river) and 7-day cumulative rainfall (rain_r7) as the dominant risk drivers. Elevation and slope contribute significantly but are secondary to hydrological factors.

ROC Curve — Spatial Block Validation

Spatial block cross-validation was used instead of random k-fold to prevent geographic data leakage. The RF model achieves AUC = 0.888 under these stricter conditions — a robust result for real-world deployment.

Key Research Findings
Random Forest outperforms XGBoost and Logistic Regression under spatial block validation (AUC 0.888 vs 0.863 vs 0.791).
SHAP ablation study confirms rain_r7 as single most critical feature — removing it drops AUC by 0.041.
River proximity (dist_river) has non-linear SHAP interaction with slope, amplifying risk at <500m from rivers on slopes >25°.
Spatial block CV reveals ~8% AUC inflation in random CV — validating the need for geographically aware model evaluation.
Flask web app with Leaflet.js delivers real-time susceptibility predictions on an interactive map interface.
